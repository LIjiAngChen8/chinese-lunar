# ✨特性
- ⏰ 可转换多种时间格式
- 🔌 提供扩展Plugin
- 🔗 支持链式操作
- 📌 可自定义假日信息
- 📦 仅 3kb 大小的轻量库
- 📑 真实可靠的中国农历规则算法，以及细化的API与对应使用文档
